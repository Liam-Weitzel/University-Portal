Tutor page:
Tutors can see their students attendance per course

Admin page:
Admin can create new courses and assign tutors to them
Admin can authorize new accounts and assign roles to them

Dashboard page:
Dashboard page is a static page explaining what the site is and how to use it etc.

Attendance page:
Attendance page shows attendance per student per course
Allows student to register online for each lecture/ seminar

Progress page:
Progress page shows progress per student per course
